**AI-Based Nuclear Power Plant Monitoring and Inspection Robot System**

An AI-powered nuclear plant inspection and monitoring system developed using Python and VPython.
The project simulates autonomous robots and drones performing real-time monitoring, hazard detection, and emergency response inside a nuclear power plant environment.

**Project Overview**
Nuclear power plants are hazardous environments that involve risks such as:
      Radiation exposure, Toxic gas leaks, Extreme temperatures
This project introduces a smart robotic inspection system that reduces human involvement by using:
     Autonomous Inspection Robot, Shielded Emergency Response Robot, Aerial Monitoring Drone
The system continuously monitors environmental conditions and automatically responds to dangerous situations.

**Features**
Real-time monitoring of:
    Temperature, Radiation, Gas levels, Pressure
Autonomous robot patrol system
Hazard detection and alert generation
Emergency robot deployment
Drone-assisted aerial monitoring
Wireless communication simulation
Battery monitoring system
Real-time graphs and dashboards
3D VPython simulation environment

**System Workflow**
Robot 1 patrols different zones of the nuclear plant
Sensor data is continuously monitored
Hazard conditions are detected using threshold values
Robot 1 retreats to a safe zone during emergencies
Robot 2 is activated for suppression and emergency handling
Drone provides aerial monitoring and communication
System resumes normal operation after conditions stabilize
